# How to Set Bot Profile Picture

## Quick Steps

1. **Save your generated image** (PNG or JPG, square format, 640x640+ pixels)

2. **Open Telegram** and search for `@BotFather`

3. **Send the command**: `/mybots`

4. **Select your bot** from the list (Crypto Market Dashboard Bot)

5. **Choose**: "Bot Settings" â†’ "Edit Botpic"

6. **Upload your image file**

7. **Confirm** when prompted

## Requirements

- **Format**: PNG or JPG
- **Size**: Minimum 640x640 pixels (recommended: 1024x1024)
- **Shape**: Square (1:1 aspect ratio)
- **File size**: Should be reasonable (Telegram may compress large files)

## Troubleshooting

- If the upload fails, try:
  - Resizing the image to exactly 1024x1024 pixels
  - Converting to PNG format
  - Reducing file size if it's too large
  - Ensuring the image is square (not rectangular)

## Your Bot Token

Your bot token is set via environment variable: `TELEGRAM_BOT_TOKEN`

To verify your bot username, you can check with BotFather using `/mybots`.
